Install

Select the appropriate version, depending on your environment (windows or mac). Initiate the install by double-clicking the icon, and follow the steps. New releases of QC Tools will be periodically available at the QC Tools project website

Load Video Files

Once the QC Tools application is open, you may identify selected video files for QC Tools analysis in three ways:
  1. Double click the folder icon on the upper left. This launches a window from which you can browse and select video files.
  2. Navigating to File --> Open.
  3. Simply drag a video file into the QC Tools window

Select Analysis Filters

By clicking the filter checkboxes you can select those that you wish to analyze and display. You may make these selections before uploading your video or at any time after the QC Tools analysis has been done and the graph display will update dynamically. As a default, 'Y values', 'U values', 'V values', and 'Diffs' are selected. For descriptions of each Filter and how to read graph values, please see the Help Section, denoted by the '?' icon in the toolbox portion of the application.

View and Navigate Graphs

Values display above, corresponding video frames show below. The video frames may be navigated via the next, previous, or playback buttons; the frame and timecode for the particular selection will be displayed.
Pointing your keyboard cursor to a particular point on a graph will result in the display of two values: the frame, and the timecode of the particular place in the video you have navigated to.
Scrubbing your keyboard cursor over the graph will cause the thumbnails to update to show the images that correspond to the associated
You may also use the '+' and '-' icons in the tool box section of the application to zoom in/out, giving you a more or less detailed view of the graph displays, depending.

Playback and Visual Analysis

By clicking on the central thumbnail of the thumbnail strip you can open the preview window. The preview window serves as a playback environment that allows spot checking and manual video analysis. The preview window contains two playback windows that can be set to various filters, the filters allow the video to be processed in one of many ways which may help me particular issues more decernable. See the Filter Description page for more details on these playback filters.

Create/ Export a CSV File

You may execute this function in one of two ways:
  1. Navigate to File --> CSV
  2. Click the 'CSV' icon in the toolbox section of the application.
Both options will prompt you to name your file and select a select a location.

Additional Functionality in Future Releases

  • The ability to re-import CSV files for analysis.
  • The ability to select/ add multiple video files to the tool.
